PHP چیست؟

PHP (مخفف PHP: Hypertext Preprocessor) یک زبان پرکاربرد، متن باز و چند منظوره اسکریپت نویسی ازت که به شکل خاص مناسب توسعه وب است و قابلیت تعبیه به درون HTML را داراست.

ایول، اما یعنی چه؟ مثال:

Example #1 یک مثال جهت معرفی

<!DOCTYPE html>
<html>
    <head>
        <title>Example</title>
    </head>
    <body>

        <?php
            echo "Hi, I'm a PHP script!";
        ?>

    </body>
</html>

به جای استفاده از دستور های فراوان برای خروجی HTML گرفتن (مثل C یا Perl)، صفحات PHP می‌توانند شامل HTML به همراه کد تعبیه شده ای باشند که "کاری" (در این مورد، خروجی دادن "Hi, I'm a PHP script!") انجام می‌دهد. کد PHP درون عملگر های مخصوص شروع و پایان پردازش <?php و ?> قرار می‌گیرد که به شما اجازه ورود و خروج به "وضعیت PHP" را می‌دهد.

چیزی که PHP را از زبان سمت کاربری مثل JavaScript متمایز می‌کند این است که کد روی سرور اجرا می‌شود، جهت تولید HTML که به کاربر فرستاده می‌شود. کاربر نتیجه اجرای اسکریپت روی سرور را دریافت خواهد کرد اما نخواهد دانست که کد مربوطه چه بوده. شما می‌توانید وب‌سرور خود را جوری تنظیم کنید که تمام فایلهای HTML را با PHP پردازش کند و در نتیجه واقعا هیچ راهی برای کاربر وجود ندارد که پی ببرد چه در آستین دارید.

بهترین چیز درباره PHP این است که هم برای یک تازه‌کار بسیار ساده است، و هم ویژگی های پیشرفته بسیاری را در اختیار برنامه نویس خبره قرار می‌دهد. از خواندن لیست بلند و بالای ویژگی های PHP نترسید. طی اندک زمانی می‌توانید وارد گود شوید و طی چند ساعت شروع به نوشتن اسکریپت های ساده کنید.

با وجود اینکه توسعه PHP متمرکز بر اسکریپت نویسی سمت سرور است، می‌توانید کارهای بیشتری نیز با آن بکنید. به خواندن ادامه دهید و از بخش PHP قادر به انجام چه کار هاییست؟ اطلاعات بیشتری کسب کنید، یا اگر علاقه مند به برنامه نویسی هستید مستقیما به آموزش مقدماتی مراجعه کنید.
