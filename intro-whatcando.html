PHP قادر به انجام چه کار هاییست؟

هر کاری. PHP عمدتا متمرکز بر اسکریپت نویسی سمت سرور است، پس می‌توانید هر کاری که سایر برنامه های CGI قادر بر آنند را انجام دهید، مثل گردآوری داده های فرم، تولید محتوای صفحات پویا یا ارسال و دریافت کوکی. اما PHP قادر به بسیار بیش از این است.

۳ حیطه وجود دارد که در آن می‌توان از PHP استفاده کرد.

  • اسکریپت نویسی سمت سرور. این سنتی ترین و اصلی ترین حیطه هدف برای PHP است. برای راه اندازی اش به ۳ چیز نیاز دارید: یک PHP parser (CGI یا ماژول سرور)، یک وب سرور و یک مرورگر وب. نیاز دارید تا وب سرور را به همراه یک نصب PHP متصل به آن اجرا کنید. شما قادرید تا به خروجی برنامه PHP با یک مرورگر وب دسترسی پیدا کرده و صفحه PHP را از طریق سرور مشاهده کنید. تمام این موارد بر روی دستگاه شخصی شما قابل اجراست اگر صرفا قصد تجربه و آزمایش برنامه نویسی PHP را دارید. جهت اطلاعات بیشتر به بخش دستورالعمل های نصب مراجعه کنید
  • اسکریپت‌نویسی خط فرمان. شما می‌تواند اسکریپتی با PHP بنویسید که بدون هیچ سرور یا مرورگری قابل اجرا باشد. برای این کار فقط به پارسر PHP نیاز خواهید داشت. این طریقه استفاده، برای اسکریپت هایی که به شکل معمول با cron (روی *نیک یا لینوکس) یا Task Scheduler (روی ویندوز) اجرا می‌شوند ایده‌آل است. این اسکریپت ها می‌توانند برای اعمال ساده پردازش متن نیز استفاده شوند. برای اطلاعات بیشتر به موارد استفاده خط‌فرمان PHP مراجعه کنید.
  • نوشتن نرم‌افزار های دسکتاپ. احتمالا PHP بهترین زبان برای ساخت یک نرم‌افزار دسکتاپ دارای رابط کاربری گرافیکی نباشد، اما اگر شما آشنایی خیلی خوبی با PHP دارید و همچنین مایل به استفاده از ویژگی های پیشرفته PHP در نرم‌افزار سمت‌کاربر خود هستید، می‌توانید از PHP-GTK نیز برای نوشتن این نوع نرم‌افزار ها استفاده کنید. علاوه بر آن از این طریق قادر به نوشتن نرم‌افزار های چند پلتفرمه نیز هستید. PHP-GTK یک افزونه PHP است و در توزیع اصلی موجود نیست. اگر به PHP-GTK علاقه مند هستید، از » وبسایت مربوطه اش بازدید کنید.

PHP قابلیت استفاده بر روی تمامی سیستم عامل های مطرح را دارد، شامل لینوکس، بسیاری از انواع یونیکس (شامل HP-UX، سولاریس و OpenBSD)، مایکروسافت ویندوز، مک، RISC OS و احتمالا تعدادی دیگر. امروزه PHP توسط بسیاری از وب‌سرور ها نیز پشتیبانی می‌شود. شامل آپاچی، IIS و بسیاری دیگر. این شامل هر وب‌سروری است که از باینری FastCGI PHP بهره بگیرد، مثل lighttpd و nginx. PHP هم می‌تواند به عنوان یک ماژول و هم به عنوان یک CGI processor کار کند.

با PHP، شما آزادید که سیستم عامل و وب‌سرور دل‌خواهتان را انتخاب کنید. علاوه بر آن شما حق انتخاب بین برنامه‌نویسی رویه‌ای (Procedural)، برنامه‌نویسی شیئ‌گرا (OOP) یا ترکیبی از هردو را دارید.

با PHP شما محدود به صرفا خروجی HTML گرفتن نیستید. توانایی های PHP شامل خروجی دادن در لحظه عکس‌ها، فایل‌های PDF و حتی کلیپ های Flash است (با استفاده از libswf و Ming). شما همچنین می‌توانید هرنوع متنی را مثل XHTML و هر نوع دیگری از فایل XML خروجی دهید. PHP می‌تواند به جای پرینت کردن، به شکل خودکار این فایل‌ها را تولید کند و در راستای محتوای پویای شما در قالب یک کَش سمت سرور در فایل‌سیستم ذخیره کند.

یکی از قوی‌ترین و چشم‌گیر ترین ویژگی ها در PHP، پشتیبانی‌اش از طیف عظیمی از پایگاه‌های داده است. نوشتن یک صفحه وب پایگاه داده محور با استفاده از یکی از افزونه‌های مختص به پایگاه‌داده (مثل mysql)، یا با استفاده از یک لایه انتزاعی (Abstraction Layer) مثل PDO، به طرز فوق‌العاده ای ساده است، همینطور اتصال به هر پایگاه‌داده ای که از استاندارد Open Database Connection از طریق افزونه ODBC استفاده می‌کند. سایر پایگاه‌های داده امکان دارد از cURL یا sockets، بهره ببرند مثل like CouchDB.

PHP همینطور از تعامل با سایر سرویس ها از طریق پروتکل‌های LDAP، IMAP، SNMP، NNTP، POP3، HTTP، COM (روی ویندوز) و بسیاری دیگر پشتیبانی می‌کند. شما همینطور می‌توانید سوکت شبکه های خامی باز کنید و با استفاده از هر پروتکل دیگری اقدام به تعامل کنید. PHP از تبادل داده پیچیده WDDX برای تقریبا تمام زبان‌های برنامه‌نویسی وب پشتیبانی می‌کند. در مورد اتصال متقابل، PHP از نمونه‌سازی (Instantiation) شیئ (Object) های جاوا و همینطور استفاده شفاف از آنها به عنوان شیئ های PHP پشتیبانی می‌کند.

PHP دارای امکانات پردازش متن پرکاربردی است، شامل عبارت‌های با قاعده (Regular Expressions) سازگار با Perl، (PCRE) و افزونه‌ها و ابزارهای بسیاری برای پردازش و دسترسی به اسناد XML. PHP تمام افزونه‌های XML را بر بستر پایدارlibxml2 استاندارد سازی می‌کند و فهرست امکانات را با اضافه کردن پشتیبانی از SimpleXML، XMLReader و XMLWriter گسترش می‌دهد.

و بسیاری افزونه‌های جالب دیگری که موجودند و بر حسب الفبا یا بر حسب دسته‌بندی گروه‌بندی شده اند. اضافه بر این، افزونه‌های PECL نیز وجود دارند که امکان دارد ذیل راهنمای PHP مستندسازی بشوند یا نشوند، مثل » XDebug.

همانطور که ملاحظه می‌کنید این صفحه برای فهرست کردن تمام ویژگی‌ها و مزایایی که PHP ارائه می‌کند کافی نیست. به خواندن در بخش های مربوط به نصبPHP ادامه دهید و سری به بخش مرجع توابع بزنید جهت خواندن توضیحات افزونه های ذکر شده در اینجا.
